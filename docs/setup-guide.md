# 🚀 Setup Guide

## Prerequisites
- Java 17+
- Maven
- Redis running (port 6379)

## Start Order

1. Eureka Server (8761)
2. Config Server (8888)
3. Auth Service (8083)
4. User Service (8091)
5. Order Service (8092)
6. API Gateway (8080)

## Access

- Eureka → http://localhost:8761
- Gateway → http://localhost:8080
